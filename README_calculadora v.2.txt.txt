# Calculadora en Python
Esta es una calculadora simple programada en Python que permite realizar operaciones matemáticas básicas.

## Instrucciones
1. Clona el repositorio.
2. Asegúrate de tener Python instalado.
3. Ejecuta el archivo `interfaz.py` para abrir la calculadora.

## Requisitos
- Python 3.x
- Tkinter (si usas una interfaz gráfica)
